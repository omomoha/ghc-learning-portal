      Master autonomous AI agents, custom skills, MCP integrations, multi-model strategy, and enterprise governance. This full-day workshop program takes developers beyond basic Copilot usage into the realm of agentic engineering — where AI agents plan multi-step tasks, execute terminal commands, generate pull requests from issues, and collaborate across workflows.

      Across two workshops and 16 hands-on modules, you will configure agent modes, craft effective prompts, build MCP servers, design reusable skills, implement automated code review pipelines, and establish enterprise-grade AI governance. Each module includes technical assessments to validate your understanding.

      Topics covered in this learning path

      • Configuring and operating GitHub Copilot's agent mode, background agents, and coding agents.
      • Crafting effective prompts and issue templates that drive autonomous multi-file changes.
      • Using the Plan Agent to architect, review, and iterate on complex code modifications.
      • Setting up Copilot Code Review (CCR) for automated PR reviews with CodeQL and ESLint integration.
      • Building custom skills with SKILL.md and the Copilot Skills Framework.
      • Developing MCP servers in Node.js and Python for AI tool integrations.
      • Implementing enterprise model strategy, AI governance, and compliance auditing.
      • Managing context windows, building custom context providers, and optimizing AI performance.
      • Creating Copilot Extensions and measuring organizational ROI.

      By the end of this learning path, you will be able to design end-to-end agentic workflows, build custom AI tool integrations, automate code review pipelines, and implement enterprise-grade governance for AI-assisted development at scale.

      Important

      To complete this training, you must have a GitHub Copilot Business or Enterprise subscription with agent mode enabled. A GitHub Copilot Free or Individual plan is not sufficient for the coding agent and background agent exercises. If your organization manages your Copilot subscription, ensure your administrator has enabled agent mode, MCP server support, and Copilot Extensions in your organization's Copilot policy settings.

      Prerequisites

      • Proficiency with GitHub Copilot Chat and basic IDE integration (code completions, inline suggestions, chat-based Q&A).
      • A GitHub Copilot Business or Enterprise subscription with agent mode enabled.
      • Visual Studio Code 1.99+ with the GitHub Copilot extension (v1.250+) installed.
      • Working knowledge of JavaScript/TypeScript and Python (code examples use both).
      • Familiarity with Git workflows, pull requests, and GitHub Actions.
      • Experience with at least one CI/CD pipeline or automated testing framework.
